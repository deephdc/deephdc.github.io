Welcome to the DEEP Open Catalog!

DEEP-Hybrid-DataCloud is a project that aims to deliver a framework to easily develop Machine Learning and Deep Learning models on top of e-Infrastructures. In the DEEP Open Catalog you can find ready to use models in a variety of domains. These models can be executed on your local laptop, on a production server or on top of e-Infrastrucutres supporting the DEEP-Hybrid-DataCloud stack.

Quickstart

1. Choose a model

Browse the list of available models and pick the one of your choice.

2. Download & execute

$ docker search deephdc
$ docker pull deephdc/deep-oc-generic-container
$ docker run -ti -p 5000:5000 deephdc/deep-oc-generic-container

3. Browse the API

  • Open your browser and visit http://127.0.0.1:5000/.
  • Read the Swagger documentation and test the model directly.
